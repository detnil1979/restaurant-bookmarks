# 我的餐廳書籤資料庫 🍜

從 Google Maps 已儲存清單整理出的 530 間餐廳書籤,提供「依需求推薦」與「瀏覽篩選」兩種模式,依種類(吃到飽/老饕/這裡很無聊)、地區、星等整理。

## 檔案說明與使用方式

| 檔案 | 用途 | 使用方式 |
|------|------|----------|
| `index.html` | 網頁版(線上開啟) | 直接開啟 <https://detnil1979.github.io/restaurant-bookmarks/>,手機、電腦瀏覽器都能用,免安裝 |
| `我的餐廳書籤-v1.1.0.apk` | Android App | 見下方安裝步驟 |
| `我的餐廳書籤資料庫-v1.1.0.html` | 網頁版(離線檔案) | 下載後直接用瀏覽器開啟,離線可用 |

## Android APK 安裝步驟

1. 手機下載 APK:[點此下載](https://github.com/detnil1979/restaurant-bookmarks/raw/main/我的餐廳書籤-v1.1.0.apk)
2. 開啟下載的檔案,若跳出「不允許安裝未知的應用程式」,依提示允許該來源安裝
3. 安裝完成後,主畫面會出現橘色「餐」字圖示的「我的餐廳書籤」
4. 需求:Android 5.0 以上;點餐廳連結會自動開啟 Google Maps

> 更新版本時直接安裝新 APK 覆蓋即可,資料內建於 App,無需備份。

## 功能

- **依需求推薦**:輸入「新莊想吃火鍋」之類的需求,自動比對種類與地區;找不到符合的會直接告知,不亂湊結果
- **瀏覽全部書籤**:依種類(吃到飽/老饕/這裡很無聊)、地區篩選,搜尋店名,星等排序
- 星等與評論數為 Google 地圖即時擷取資料,僅供個人參考
