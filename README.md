# 我的餐廳書籤資料庫 🍜

從 Google Maps 已儲存清單整理出的 530 間餐廳書籤,提供「依需求推薦」與「瀏覽篩選」兩種模式,依種類(吃到飽/老饕/這裡很無聊)、地區、星等整理。另附 84 間「路人推薦」與 178 間「Threads推薦」,並與個人書籤分開標示。

## 檔案說明與使用方式

| 檔案 | 用途 | 使用方式 |
|------|------|----------|
| `index.html` | 網頁版(線上開啟) | 直接開啟 <https://detnil1979.github.io/restaurant-bookmarks/>,手機、電腦瀏覽器都能用,免安裝 |
| `我的餐廳書籤-v1.4.0.apk` | Android App | 見下方安裝步驟 |
| `我的餐廳書籤資料庫-v1.5.0.html` | 網頁版(離線檔案) | 下載後直接用瀏覽器開啟,離線可用 |
| `data/threads_taipei_restaurants_2026-07-21.csv` | Threads 推薦來源資料 | 178 筆店名、類型、地點、推薦人與留言連結 |

## Android APK 安裝步驟

1. 手機下載 APK:[點此下載](https://github.com/detnil1979/restaurant-bookmarks/raw/main/我的餐廳書籤-v1.4.0.apk)
2. 開啟下載的檔案,若跳出「不允許安裝未知的應用程式」,依提示允許該來源安裝
3. 安裝完成後,主畫面會出現橘色「餐」字圖示的「我的餐廳書籤」
4. 需求:Android 5.0 以上;點餐廳連結會自動開啟 Google Maps

> v1.2.0 之後的版本直接安裝覆蓋即可(同簽章)。仍在 v1.1.0 者:因 v1.2.0 起重新簽章,請先解除安裝舊版再安裝(資料內建於 App,無需備份)。

## 功能

- **依需求推薦**:輸入「新莊想吃火鍋」之類的需求,自動比對種類與地區;找不到符合的會直接告知,不亂湊結果
- **瀏覽全部書籤**:地區分組列表(黏性標題+右側快速跳轉),種類一鍵篩選(全部/吃到飽/老饕/這裡很無聊/路人推薦/Threads推薦),搜尋店名、類型、地點或推薦人,星等/名稱排序;點任一列展開完整資訊(星等評論、地址、筆記/備註、Google 地圖連結)
- **路人推薦**:另外收集的口袋名單(Google 試算表),以紫色「路人推薦」標籤與個人書籤分開;瀏覽頁可用種類「路人推薦」單獨篩選,依需求推薦也會一併比對
- **Threads推薦**:2026-07-21 整理的 178 筆 Threads 留言推薦,保留推薦人、推薦重點、原留言摘錄與原留言連結;以藍色標籤獨立篩選
- **熱門食記**:每間店展開後有食記按鈕——比對得到的連到愛食記店頁(彙整多篇食記與評分),其餘為 Google 食記搜尋
- 星等與評論數為 Google 地圖即時擷取資料,僅供個人參考

## 路人推薦資料更新

路人推薦清單來自 Google 試算表「台北餐廳清單_補齊地址」。App 每次開啟會嘗試讀取試算表最新內容,讀不到時使用內建快照(2026-07-16)。

要啟用「編輯試算表 → App 自動更新」:把該試算表的共用設定改為「知道連結的使用者皆可檢視」即可,無需改程式。試算表保持私人時,App 仍可正常使用內建快照。
