# 純粹吃東西 🍜

從 Google Maps 已儲存清單整理出的餐廳工具，提供「幫我選」與「找餐廳」兩種模式。可在自己的裝置保存心得、照片與隱藏清單，也能分享單間餐廳、個人心得或整個 App。

## 檔案說明與使用方式

| 檔案 | 用途 | 使用方式 |
|------|------|----------|
| `index.html` | 網頁版(線上開啟) | 直接開啟 <https://detnil1979.github.io/restaurant-bookmarks/>,手機、電腦瀏覽器都能用,免安裝 |
| `純粹吃東西-v1.8.0.apk` | Android App | 見下方安裝步驟 |
| `純粹吃東西-v1.8.0.html` | 網頁版(離線檔案) | 下載後直接用瀏覽器開啟,離線可用 |
| `data/threads_taipei_restaurants_2026-07-21.csv` | Threads 推薦來源資料 | 178 筆原始推薦與補齊欄位（標準分類、Google 店名/地址/地區/連結/評分/價位/類別、查證狀態） |

## Android APK 安裝步驟

1. 手機下載 APK:[點此下載](https://github.com/detnil1979/restaurant-bookmarks/raw/main/純粹吃東西-v1.8.0.apk)
2. 開啟下載的檔案,若跳出「不允許安裝未知的應用程式」,依提示允許該來源安裝
3. 安裝完成後,主畫面會出現橘色「純」字圖示的「純粹吃東西」
4. 需求:Android 5.0 以上;點餐廳連結會自動開啟 Google Maps

> v1.2.0 之後的版本直接安裝覆蓋即可(同簽章)。安裝 v1.8.0 後,未來開啟 App 時會自動檢查新版並提供下載按鈕。

## 功能

- **幫我選**:輸入「新莊想吃火鍋」之類的需求,自動比對種類與地區;已隱藏的店不會再出現在推薦中
- **隨便**:大型按鈕每按一次就自動換一間不同餐廳;連續第 5 次會提醒「再挑就餓死了，隨便啦」
- **找餐廳**:清楚標示搜尋、地區與排序,手機底部提供固定快速操作;點任一列即可查看完整資訊
- **版本更新**:Android App 開啟時從 GitHub Pages 檢查最新版本;有新版會顯示更新內容、立即更新與稍後提醒
- **我的紀錄**:每間餐廳可儲存心得、公開照片連結或本機照片;文字與隱藏清單存於 localStorage,本機照片存於 IndexedDB
- **分享**:支援分享單間餐廳、包含心得的分享連結，以及 App 分享;不支援 Web Share API 時會自動複製連結
- **隱藏餐廳**:可隱藏不想再看到的餐廳並記住設定,「已隱藏」篩選可查看與恢復
- **路人推薦**:另外收集的口袋名單(Google 試算表),以紫色「路人推薦」標籤與個人書籤分開;瀏覽頁可用種類「路人推薦」單獨篩選,依需求推薦也會一併比對
- **Threads推薦**:2026-07-21 整理的 178 筆 Threads 留言推薦,保留推薦人、推薦重點、原留言摘錄與原留言連結;140 筆已對應 Google Maps 並補齊 app 標準欄位,38 筆因缺少分店或地址標為待確認
- **熱門食記**:每間店展開後有食記按鈕——比對得到的連到愛食記店頁(彙整多篇食記與評分),其餘為 Google 食記搜尋
- 星等與評論數為 Google 地圖即時擷取資料,僅供個人參考

## 路人推薦資料更新

路人推薦清單來自 Google 試算表「台北餐廳清單_補齊地址」。App 每次開啟會嘗試讀取試算表最新內容,讀不到時使用內建快照(2026-07-16)。

要啟用「編輯試算表 → App 自動更新」:把該試算表的共用設定改為「知道連結的使用者皆可檢視」即可,無需改程式。試算表保持私人時,App 仍可正常使用內建快照。
